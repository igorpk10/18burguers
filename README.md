# Dependências
- Java 17
- Maven 3.6.3+


## Instalar dependências
```
mvn clean install
```

## Construir o Projeto
```
mvn clean package
```

## Executar Aplicação
```
mvn spring-boot:run
```

## Construir infraestrutura
```
docker-compose up -d 
```

## Executar Testes
```
mvn test
```
